Упражнение 15. Моделирование броуновского движения частиц

Требуется разработать модель броуновского движения N частиц в квадратной области с размером стороны в а единиц. Частицы считать абсолютно твердыми телами круглой формы с радиусом r единиц. Начальные значения координат и скоростей частиц являются случайными и соответствуют равномерному закону распределению. Координаты центров частиц лежат в интервале от r до a-r. Компоненты вектора начальной скорости распределены в интервале от -V0 до V0. В исходном состоянии частицы не перекрываются и не пересекаются со стенками области. Столкновения частиц между собой и со стенками области считать абсолютно упругими.

Исходные данные:
Размер области a = 100;
Радиус частицы r = 2;
Число частиц N = 40;
Граница интервала по начальной скорости V0 = 5.
F Пример выполняемой программы E

Пояснения к алгоритму

Алгоритм имитации броуновского движения может включать следующие шаги: 

  1. Заполнить область частицами со случайными параметрами
  2. Установить начальные значения времени t процесса 
     и шага по времени dt (Например: t=0; dt=0.08)
  3. Организовать цикл по времени для имитации движения
     isStop:=False
     while ( isStop=False )
        t:=t+dt
        for k=1 to N step 1
         вычисление приращения координат k-ого шара: dx,dy
         проверка столкновений:
            Ok:=True
            while (Ok=True)
              Ok:=False
              if есть столкновение с шарами
                then 
                  корректировка dx,dy,Vx,Vy 
                  Ok:=False
              end if
              if есть столкновение со стенкой
                then 
                  корректировка dx,dy,Vx,Vy 
                  Ok:=True
              end if
            end while
         стирание k-ого шарика
         установка новых значений координат k-ого шарика
         рисование k-ого шарика в новом положении
        next k
        проверка остановки счета 
     end while
Генерация начального распределения шаров 

  for k=1 to N step 1
    Ok:=True
    while (Ok=True)
      определение случайных координат k-ого шара: 
       x:=2*r+Random*(a-4*r)
       y:=2*r+Random*(a-4*r)
      проверка на пересечение с ранее сгенерированными шара: 
       Ok:=True
       for j=1 to k-1 step 1
        if есть пересечение j-ого и k-ого шаров
           then Ok:=False
       next j
    end while
  next k
